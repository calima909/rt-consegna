Red Tetris - Applicazione Web Multiplayer in tempo reale

Progettazione e sviluppo di un gioco Tetris multiplayer utilizzando JavaScript. Realizzata un’architettura client-server con frontend Vue e backend Node.js responsabile della gestione dello stato di gioco, dei giocatori e della sincronizzazione in tempo reale tramite WebSocket.

Implementato un sistema deterministico di generazione dei pezzi per garantire coerenza tra tutti i client e una meccanica competitiva basata sull’invio di penalità agli avversari. Applicati principi di programmazione funzionale lato client (pure functions, stato immutabile) e design object-oriented lato server.
